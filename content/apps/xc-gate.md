# XC-Gate（エクシーゲート） - 現場帳票ペーパーレス化ソリューション 調査報告

> **ビジョン：手書き帳票をゼロにし、現場データをリアルタイムで経営に活かす**

---

## 1. 現在の課題（Problem）

製造現場・各種業務拠点で日常的に発生している紙帳票課題を整理しました。

### 1-1. 手書き帳票の収集・転記に膨大な工数がかかる

日報・検査表・チェックシートなど、現場で手書きされた帳票は、後から管理者やスタッフが内容を読み取って基幹システムへ手入力するという二重作業が発生している。

ある製造業の導入事例では、毎日80〜90枚提出される作業日報を管理者5名が基幹システムへ入力する作業だけで、**1日のべ4.5時間以上**を要していた。

### 1-2. 紙の保管・検索・集計が非効率

手書き帳票は物理的な保管スペースを消費し、過去データの検索や集計に時間がかかる。品質異常や監査対応時に過去の検査記録を素早く参照できない。

```
現状の帳票フロー:
├── 現場作業員      ← 紙に手書き記入
├── 管理者収集      ← 紙を集めて確認・押印
├── 担当者転記      ← PCで基幹システムへ手入力（二重作業）
├── 紙保管          ← ファイリング・保管スペース消費
└── → 集計・分析は別途Excelで手作業
```

### 1-3. リアルタイムでのデータ把握ができない

紙ベースでは、現場の状況を管理者がリアルタイムで把握することが不可能。問題発生時の対応遅れ、品質トレンドの見落としが生じやすい。

## 2. XC-Gateでこう変わる（Solution）

| # | 現状（BEFORE） | 導入後（AFTER） |
|---|---|---|
| 1 | 現場で手書き → 管理者が基幹システムへ転記 | **タブレット直接入力**で転記作業がゼロに |
| 2 | Excelで帳票を管理・改修にIT部門が必要 | **Excel設計そのまま電子化**、現場担当者が自分で改修可能 |
| 3 | 紙を集めてから集計（タイムラグ大） | 入力と同時に**データベース蓄積**、リアルタイム集計・分析 |
| 4 | 紙の承認フローで回覧・押印 | **電子ワークフロー**で承認・電子サインが完結（今後活用） |

### XC-Gate導入後の帳票管理構成

```
XC-Gate
└── 帳票マスタ（Excelで設計）
    ├── 入力フォーム（タブレット/スマホ/PC対応）
    │   ├── テキスト・数値・日時・ドロップダウン入力
    │   ├── カメラ撮影・バーコード読取
    │   └── オフライン入力（ネットワーク不要）
    ├── データ管理
    │   ├── DBへ自動蓄積（即時検索可能）
    │   ├── PDF出力・帳票印刷
    │   └── CSV出力・外部システム連携
    └── ワークフロー
        ├── 承認ルート設定
        └── 電子サイン
```

Excelで設計した帳票レイアウトがそのままタブレット画面に変換されるため、現場担当者が使い慣れた帳票を維持しながらペーパーレスに移行できます。

## 4. 主要機能の詳細

### 4-1. Excel帳票の電子化

既存のExcel帳票をそのままWebフォームに変換。Excel関数・条件付き書式・入力規則もそのまま反映される。IT部門に頼らず現場の担当者自身が帳票を修正・追加できる。

```
入力: 既存Excelファイル（日報・検査表・チェックシートなど）
  ↓
処理: XC-Gate画面設計ツールでWebフォームに変換
  ↓
出力: タブレット/PCで入力できる電子帳票（元のレイアウトを維持）
```

### 4-2. マルチデバイス対応・オフライン入力

PC・タブレット・スマートフォン・ハンディターミナルなど様々なデバイスに対応。ネットワークが届かない現場でもオフラインで入力でき、通信回復後に自動同期する。

### 4-3. データ収集・集計・分析

入力データはリアルタイムでデータベースに蓄積。検索・絞り込み・CSV出力が即座に可能。BIツールや基幹システムとの連携により、現場データを経営分析に活用できる。

```
現場入力（タブレット）
  ↓
DB自動蓄積
  ↓ ↓ ↓
┌─────┐ ┌─────────┐ ┌──────────┐
│CSV出力│ │BIツール連携│ │基幹システム│
└─────┘ └─────────┘ └──────────┘
```

### 4-4. ワークフロー・電子承認

承認ルートを設定することで、紙の回覧・押印フローを電子化。スマートフォンで外出先からも承認操作が可能。承認状況はリアルタイムで確認できる。
